Fase 3: Pengembangan Prototipe

Mengubah konsep abstrak menjadi solusi nyata. Di fase ini, kita memvalidasi ide BlueGuard melalui perancangan arsitektur data dan visualisasi antarmuka pengguna untuk memantau pengasaman laut secara presisi.

Metode Starbursting 4W1H

Sebelum memulai desain teknis, kami menerapkan analisis Starbursting untuk membedah setiap aspek kritis dari prototipe BlueGuard. Fokus utama kami adalah memastikan cakupan fungsionalitas yang komprehensif:

  • verified
    WhoPeneliti, Dinas Lingkungan Hidup DKI Jakarta, komunitas nelayan pesisir, dan akademisi yang membutuhkan alat prediksi berbasis data.
  • inventory_2
    WhatModel machine learning berbasis scikit-learn untuk memprediksi persentase probabilitas pencemaran nutrisi penyebab pengasaman laut berdasarkan variabel lingkungan.
  • location_on
    WhereDifokuskan pada pesisir dan daerah aliran sungai (DAS) Teluk Jakarta, dengan pengembangan model yang berjalan secara lokal.
  • schedule
    WhenUntuk memungkinkan deteksi dini dan intervensi cepat terhadap limpasan nutrisi yang sulit dipantau secara manual, guna mendukung target SDG 14.3.
  • settings
    HowMelatih model Random Forest atau Logistic Regression menggunakan data sintetis, mengujinya dengan cross-validation, lalu menampilkan probabilitas pencemaran pada terminal teks.

BlueGuard Business & Impact Model

Lean Canvas ini memetakan bagaimana aplikasi BlueGuard menciptakan nilai bagi komunitas maritim dan memastikan keberlanjutan proyek dalam jangka panjang.

Masalah

• Pengasaman laut meningkat akibat limbah industri, pertanian, dan emisi CO₂. 

• Sulit mendeteksi pencemaran laut secara dini. 

• Kurangnya sistem berbasis data untuk kebijakan lingkungan. 

• Pemantauan lingkungan masih manual dan lambat.

Solusi

• Membuat model Machine Learning berbasis scikit learn untuk prediksi risiko pengasaman laut. 

• Dashboard visualisasi hasil prediksi secara real-time. 

• Sistem klasifikasi risiko rendah hingga kritis. 

• Sistem peringatan dini berbasis AI.

Nilai Unik

• Sistem mampu memprediksi probabilitas risiko berdasarkan variabel lingkungan. 

• Fitur simulasi skenario (what-if simulation). 

• Menampilkan kontribusi setiap faktor lingkungan.

• Menggabungkan AI, visualisasi, dan simulasi interaktif.

Keunggulan

• Prediksi lebih cepat dibanding pemantauan manual. 

• Dapat dikembangkan dengan data sensor real time. 

• Scalable untuk wilayah lain. 

• Potensi integrasi dengan sensor IoT dan drone.

Segmen

• Dinas Lingkungan Hidup dan pemerintah daerah. 

• Peneliti lingkungan dan akademisi. 

• Komunitas nelayan dan organisasi lingkungan. 

• Industri pengelolaan limbah.

Metrik Kunci

• Akurasi model Machine Learning (F1-score dan AUC ROC). 

• Jumlah prediksi yang berhasil dilakukan sistem. 

• Kecepatan sistem dalam memberikan hasil prediksi risiko. 

• Tingkat ketepatan klasifikasi risiko. 

• Jumlah pengguna atau instansi yang menggunakan sistem. 

• Penurunan tingkat pencemaran berdasarkan rekomendasi sistem.

“Platform AI Prediksi Risiko Pengasaman Laut berbasis Machine Learning untuk membantu deteksi dini pencemaran lingkungan pesisir secara cepat, akurat, dan berbasis data.” 

Saluran

Channels Direct & Government Collaboration 

• Sosialisasi kepada komunitas nelayan dan organisasi lingkungan.

• Kolaborasi dengan universitas dan lembaga penelitian lingkungan. 


Digital & Educational Campaign 

• Publikasi hasil analisis dan visualisasi data melalui website interaktif. 

• Edukasi media sosial tentang dampak pengasaman laut. 

• Seminar dan workshop penggunaan AI untuk lingkungan.

Struktur Biaya

• Biaya pengembangan model Machine Learning dan backend. 

• Biaya dashboard UI/UX dan visualisasi data. 

• Biaya server cloud, database, dan hosting. 

• Biaya integrasi sensor lingkungan dan perangkat IoT. 

• Biaya pengumpulan dan pemrosesan data lingkungan. 

• Biaya pemeliharaan sistem dan pembaruan model AI. 

Aliran Pendapatan

• Kerja sama proyek dengan pemerintah daerah dan kementerian lingkungan. 

• Hibah penelitian dan pendanaan sustainability/SDGs. 

• Lisensi penggunaan platform untuk lembaga penelitian atau industri. 

• Konsultasi analisis data lingkungan berbasis AI. 

• Kerja sama dengan perusahaan industri untuk monitoring dampak lingkungan.

Panduan Operasional Prototipe

input

1. Input Data Lingkungan

Pengguna memasukkan parameter lingkungan (curah hujan, vegetasi, volume pupuk, dll.) ke dalam sistem untuk memulai pemodelan risiko.

analytics

2. Hasil Prediksi

Sistem menyajikan angka probabilitas pencemaran secara real-time beserta rincian persentase kontribusi dari masing-masing variabel.

history

3. Riwayat Prediksi

Dasbor perekaman jejak yang menyimpan seluruh aktivitas prediksi sebelumnya, lengkap dengan waktu dan pelabelan status risikonya.

handshake

4. Simulasi Skenario

Fitur what-if interaktif yang memungkinkan pengguna memanipulasi variabel untuk melihat simulasi dampak penurunan risiko pencemaran.
